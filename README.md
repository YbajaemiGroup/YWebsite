# YWebsite
This is website for ybajaemi clan

Вся документация расположена в обменнике Mortar.

Структура проекта:
```
ybajaemi
|
|___ control
|
|___ website
      |
      |___ core
      |
      |___ front
```
Control - папка с программой для управления сайтом

Website - сам сайт, внутри него:

front - все, что связано с самим сайтом

core - это сервер
